Introduction¶

  1. Data Visualization is the process of presenting data in the form of graphs or charts.
  2. It helps to understand large and complex amounts of data very easily.
  3. It allows us to identify new trends and patterns very easily.

Matploptib is a low-level library of Python which is used for data visualization, This library is built on the top of NumPy arrays and consist of several plots like line chart, bar chart, histogram, etc.These various plots we can be visualized using Pyplot
